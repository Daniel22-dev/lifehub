# LifeHub 4.8.0 – bezpečná aktualizace

1. V současném LifeHubu vytvořte **kompletní šifrovanou zálohu** a ověřte ji.
2. Nahrajte celý obsah tohoto balíku do kořene stejného GitHub repozitáře.
3. Počkejte na zelený běh **Deploy LifeHub to GitHub Pages**.
4. Otevřete původní nainstalovaný LifeHub a potvrďte nabídnutou aktualizaci.
5. Odemkněte aplikaci původním hlavním heslem nebo dříve aktivovaným zabezpečením telefonu.
6. Ověřte verzi **4.8.0** v patičce.
7. Zkontrolujte zejména:
   - období Odměn,
   - náhled AI výkazu,
   - propojení nové uhrazené platby s finančním výdajem,
   - mzdové období a datum připsání u nové výplatní pásky.
8. Po ověření vytvořte novou kompletní zálohu.

Aktualizace nemění hlavní heslo, nemaže lokální trezor a zachovává biometrické rychlé odemčení. Starší historie úhrad se automaticky zpětně nepřepisuje do financí, aby nevznikly duplicitní výdaje; propojení se použije u nových úhrad a dalších automatických termínů.
