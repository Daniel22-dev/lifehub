# LifeHub 4.8.5 – stručný návod po aktualizaci

1. Nahraj obsah ZIPu do kořene stejného GitHub repozitáře.
2. Po dokončení GitHub Actions otevři LifeHub a potvrď nabídku nové verze.
3. Použij stejné hlavní heslo jako dosud.
4. Pokud je biometrika aktivní, systémový dialog se má otevřít automaticky.
5. Na Přehledu zkontroluj položku **Připsaná výplata**. Starší červnová páska s odhadovaným datem se má započítat do července.
6. Klepni na tlačítko celé obrazovky v horní liště. Telefon má skrýt systémové prvky; pokud to odmítne, LifeHub zapne záložní rozšířený režim.
7. Po kontrole vytvoř kompletní šifrovanou zálohu.

Aktualizace nemaže lokální data ani nemění hlavní heslo.
