# LifeHub 5.0.3

## Viditelné potvrzení koupeného nákupu

- Po kliknutí se v zaškrtávacím políčku nejprve zobrazí fajfka.
- Karta krátce zezelená a text se změní na „Koupeno“.
- Teprve po 650 ms se položka přesune mezi koupené.
- Finanční výdaj se zapisuje a šifrovaně ukládá okamžitě, takže vizuální prodleva neodkládá práci s daty.
- Políčko je zvětšené pro lepší ovládání na telefonu.
