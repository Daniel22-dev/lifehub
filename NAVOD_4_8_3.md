# Aktualizace na LifeHub 4.8.3

1. V současné aplikaci vytvořte kompletní šifrovanou zálohu a ověřte ji bez importu.
2. Nahrajte obsah ZIPu do kořene stejného GitHub repozitáře.
3. Počkejte na zelený CI i nasazení GitHub Pages.
4. LifeHub otevřete na stejné adrese a použijte původní hlavní heslo.
5. V interaktivním manuálu ověřte verzi 4.8.3.

Verze 4.8.3 nemění datové schéma ani šifrování. Aktualizuje především manuál a dokumentaci současných funkcí.
