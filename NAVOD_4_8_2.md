# Aktualizace na LifeHub 4.8.2

1. V LifeHubu vytvořte kompletní šifrovanou zálohu.
2. Rozbalte ZIP a nahrajte jeho obsah do kořene stejného GitHub repozitáře.
3. Počkejte na zelený deploy GitHub Pages.
4. V aplikaci potvrďte nabídnutou aktualizaci a ověřte verzi 4.8.2.
5. Dokončené úkoly, koupené/odložené nákupy a doplacené splátky najdete dole v rozbalovacích archivech.
