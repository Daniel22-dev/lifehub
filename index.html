Zálohy

Přenos telefonu ↔ PC

Kompletní záloha přibalí skutečná PDF a dokumenty z IndexedDB. Datová záloha je lehčí a přenáší jen stav aplikace a metadata. Nešifrovanou datovou zálohu nepoužívej pro přenos citlivých dat.

Ověření zálohy soubor jen načte, případně dešifruje, ukáže počty a nic nepřepíše. Import nejdřív nabídne zálohu aktuálního zařízení, ukáže náhled rozdílů a vyžádá textové potvrzení IMPORTOVAT.

Notion

Balíček pro Notion

Pro vlastní soukromý Notion použijte Markdown pro dlouhé poznámky a CSV pro tabulkové databáze. Přímé API propojení vyžaduje bezpečný backend/proxy, ne veřejně uložený API token v HTML.

Anonymizace

Export pro veřejný GitHub nebo sdílení bez originálů

Anonymizovaný export neobsahuje PDF, soubory z šifrovaného trezoru, názvy souborů, URL, extrahovaný raw text, poznámkové obsahy ani jména zaměstnavatelů. U financí používá kvartály a finanční rozsahy.

Pozor: pokud je export veřejně vedle aplikace s vaším jménem v patičce, nejde o dokonalou anonymitu, ale o výrazné odstranění identifikátorů.

Nápověda

Jak LifeHub používat

Praktický workflow

1. AI vlákna

Po dobré odpovědi z ChatGPT/Claude ulož název, stručné shrnutí, tagy, použitelný prompt a další krok. Poznámku pak otevřeš klikem a přečteš celý obsah.

2. Výplatní pásky

Ve Financích otevři čtečku PDF (tlačítko dole), zkontroluj rozpoznané hodnoty a ulož. Čistá mzda se zapíše jako příjem daného měsíce.

3. Šifrovaný trezor

Soukromé dokumenty nahraj v části Archiv. Kapacitu úložiště vidíš nahoře. Do GitHubu nepatří; mezi telefonem a PC je přenese pouze kompletní šifrovaná záloha.

4. Úkoly a nákupy

U úkolů odděl prioritu (jak to hoří) a časový horizont (kdy). U nákupů rozliš dům/velké od potravin.

5. Splátky

V sekci Splátky si nastav, komu kolik splácíš. Aplikace hlídá, kolik zbývá, a jednou týdně to připomene.

6. Aplikace

V sekci Aplikace si drž přehled projektů. Ke každé aplikaci přidáš přes + samostatné poznámky.

Jak LifeHub funguje s daty

Lokální trezor

Telefon i PC mají vlastní lokální úložiště. Data se sama nesynchronizují a aplikace neposílá obsah trezoru na server.

Heslo

Heslo se neukládá. Když ho zapomeneš, data obnovíš pouze ze šifrované zálohy a se správným heslem k této záloze.

Aktualizace

Aktualizace kódu přes GitHub Pages nemá mazat localStorage ani IndexedDB. Před větší aktualizací je ale rozumné stáhnout kompletní zálohu.

Ověření zálohy

Před důležitým importem nebo po vytvoření větší zálohy použij Ověřit zálohu bez importu. Soubor se zkontroluje, ale aktuální data se nepřepíšou.

Přenos telefon ↔ PC

Telefon jako hlavní zařízení

Na telefonu vytvoř kompletní šifrovanou zálohu. Soubor přenes do PC bezpečným způsobem a v LifeHubu na PC použij Import zálohy.

Kdy stačí datová záloha

Datová záloha je vhodná pro rychlé přenesení poznámek, financí, úkolů a metadat. Nepřenese PDF výplatních pásek ani dokumenty z archivu.

Bezpečný postup

Na cílovém zařízení nejdřív ověř zálohu bez importu. Při importu zkontroluj datum, zařízení a počty položek. Pokud aplikace varuje, že je soubor starší, import dobře zvaž.

Nejdůležitější pravidlo

Před importem si stáhni kompletní zálohu aktuálního zařízení. Když se spleteš, máš možnost návratu.

Omezení PDF importu

Aplikace čte textovou vrstvu PDF. Pokud zaměstnavatel posílá sken nebo obrázek v PDF, je nutné nejdříve použít OCR. PDF import čte jen prvních několik stran a má bezpečnostní limity. Rozpoznané částky jsou návrh; před uložením je zkontrolujte.
