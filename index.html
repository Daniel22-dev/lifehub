Zálohy

Přenos telefonu ↔ PC

Kompletní záloha přibalí skutečná PDF a dokumenty z IndexedDB. Datová záloha je lehčí a přenáší jen stav aplikace a metadata. Nešifrovanou datovou zálohu nepoužívej pro přenos citlivých dat.

Ověření zálohy soubor jen načte, případně dešifruje, ukáže počty a nic nepřepíše. Import nejdřív nabídne zálohu aktuálního zařízení, ukáže náhled rozdílů a vyžádá textové potvrzení IMPORTOVAT.

Notion

Balíček pro Notion

Pro vlastní soukromý Notion použijte Markdown pro dlouhé poznámky a CSV pro tabulkové databáze. Přímé API propojení vyžaduje bezpečný backend/proxy, ne veřejně uložený API token v HTML.

Anonymizace

Export pro veřejný GitHub nebo sdílení bez originálů

Anonymizovaný export neobsahuje PDF, soubory z šifrovaného trezoru, názvy souborů, URL, extrahovaný raw text, poznámkové obsahy ani jména zaměstnavatelů. U financí používá kvartály a finanční rozsahy.

Pozor: pokud je export veřejně vedle aplikace s vaším jménem v patičce, nejde o dokonalou anonymitu, ale o výrazné odstranění identifikátorů.

Rodina

Uložené rodinné heslo

Zatím nenastaveno

Při prvním vytvoření nebo načtení rodinného souboru ho zadáte jednou.

Bez opakovaného zadávání: heslo je součástí šifrovaného trezoru tohoto zařízení. Automatické zamknutí LifeHubu po 15 minutách ho neodstraní; po odemčení hlavního trezoru se znovu použije automaticky.
